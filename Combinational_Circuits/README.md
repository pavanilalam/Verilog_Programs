# Combinational Circuits

This folder contains Verilog HDL implementations of basic combinational logic circuits.

## Programs

- 2:1 Multiplexer
- 4:1 Multiplexer
- 1:2 Demultiplexer
- 2:4 Decoder
- 4:2 Encoder
- 4:2 Priority Encoder
- Half Adder
- Full Adder
- 4-bit Ripple Carry Adder
- 4-bit Comparator
