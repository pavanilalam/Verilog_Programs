# 4:1 Multiplexer

A 4:1 multiplexer selects one of four input signals (`i0`, `i1`, `i2`, or `i3`) based on the 2-bit select signal (`sel`).

- `mux4x1.v` ‚Äì Verilog design

## Truth Table

| sel | Output |
|-----|--------|
| 00  | i0 |
| 01  | i1 |
| 10  | i2 |
| 11  | i3 |
